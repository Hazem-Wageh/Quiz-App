# 🧠 HTML Quiz App

## 📌 Overview

This project is a fully responsive **HTML Quiz Application** that dynamically loads questions from a **JSON file**.
It is designed to demonstrate front-end skills such as data fetching, DOM manipulation, and user interaction.

---

## 🚀 Features

* 📝 9 HTML questions loaded dynamically from JSON
* 🔄 Dynamic question rendering
* ✅ Multiple choice answers
* 🎯 Score calculation system
* 🔁 Restart quiz functionality
* 📱 Fully responsive design (Mobile / Tablet / Desktop)
* ⚡ Clean and interactive UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox / Grid)
* JavaScript (ES6)
* JSON (for storing questions)

---

## 👨‍💻 My Role

As a **Front-End Developer**, I was responsible for:
